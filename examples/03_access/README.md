## Access Examples

This directory is intentionally empty for now.

Old access examples were removed because they were not trustworthy enough to
serve as verified endpoint evidence.

Add new files here only when they satisfy the branch-example contract:

- one concrete verified branch per runnable Python file
- plots saved under `plots/`
- machine-readable output, preferably a final `RESULT_JSON=...` line
- behavior backed by the internal dossier for `/access/AccessComputeV2`
